# 每日科技动态发布到 VuePress

## 目标
每天生成一篇适合网页阅读的中文科技动态文章，内容覆盖以下三个方向且尽量均衡：
- 前端
- DevOps
- AI

文章发布到用户的 VuePress 仓库后，最终响应只做两件事：
1. 给出已上线的站点地址
2. 附带一段简短新闻简讯

## 仓库与路径
默认仓库：`/Users/acongm/code/github/vuepress`

默认文档目录：`docs/daily-news/`

默认文件命名：
- Markdown：`docs/daily-news/YYYY-MM-DD.md`
- 站点 URL：`https://acongm.github.io/vuepress/daily-news/YYYY-MM-DD.html`

## 内容要求

### 选题范围
优先检索最近 24 小时；不足时可放宽到最近 72 小时，再不足时可补最近 7 天的重要更新。

### 类别平衡
- 三个类别都必须出现
- 默认每类 2 条左右
- 总计 6 条左右
- 若某类当天官方信息较少，可降到 1 条，但不要让另外两类明显失衡

### 信息筛选标准
优先保留：
- 官方发布
- 版本更新
- 新能力上线
- 生态变化
- 影响开发者工作流的消息
- 有明确来源与链接的内容

避免：
- 标题党
- 无明确来源的二手转述
- 纯营销软文
- 重复堆砌事实

## 抓取受阻时的降级策略
有些官方站点会遇到正文提取失败、Cloudflare 或抽取不稳定的情况。按下面顺序降级：
1. 优先保留官方 URL。
2. 先用 `web_search` 获取官方结果摘要。
3. 必要时用 `browser_navigate` / `browser_snapshot` 读正文。
4. 可以优先使用官方 RSS / feed 页面拿发布日期、标题、摘要。
5. 如果只能拿到官方搜索摘要，也可以据此写入，但不要补写摘要里没有的事实。

## 写作风格：网页文章，不是会议稿

### 必须避免的表达
不要使用这类会议/演讲场景导向的话术：
- “会议摘要版”
- “适合 1~2 分钟开场”
- “科技新闻会议分享稿”
- “会议上可以这样讲”
- “会上可直接抛出的讨论问题”
- “如果还要继续转成 PPT”

### 推荐表达方式
要写成通用文章风格，适合网页阅读、收藏、转发：
- 标题使用“每日科技动态”“今日科技观察”“每日科技简报”等通用名称
- 开头先给今日总观察
- 每个分类先写一个简短判断句
- 每条新闻采用“来源 + 链接 + 核心信息 + 分析说明”的结构
- 结尾给“今日观察”或“简讯”即可

### 建议结构
1. Frontmatter
   - `title`
   - `date`
   - `tags`
2. 页面标题
3. 开头总判断（2~4 句）
4. `## 今日概览`
5. `### 前端`
6. `### DevOps`
7. `### AI`
8. `## 今日观察`
9. `## 简讯`

### 文风要求
- 简体中文
- 信息密度高
- 不写空话
- 不使用演讲备注、主持词、口播提示
- 分析必须紧贴来源事实，不得脑补未核实细节

## VuePress 集成步骤
1. 在仓库中写入 `docs/daily-news/YYYY-MM-DD.md`
2. 更新 `docs/.vuepress/config.ts` 中的 navbar / sidebar，把新日期页加到 `每日资讯` 里，最新日期放前面
3. 如果仓库里有对应的索引页，也同步补充链接
4. 运行：
   - `npm run lint`
   - `npm run build`
5. 确认构建成功
6. 提交并推送到远程仓库
7. 计算最终站点地址

## Git 注意事项
- 只提交本次相关文件
- 避免把无关文件一起提交
- 如果推送被拒绝，先 `git pull --rebase` 再推送
- 如果仓库远程地址提示 moved，按新地址继续即可

## 定时任务最终响应格式
定时任务完成后，不要输出一大段过程说明。最终响应应尽量简洁，格式如下：

第一行：站点地址
例如：
`站点地址：https://acongm.github.io/vuepress/daily-news/2026-05-02.html`

然后空一行，再给“新闻简讯”：
- 前端：一句话
- DevOps：一句话
- AI：一句话

必要时可在最前面补一句：
`已发布今日科技动态。`

## 质量检查
完成前确认：
- 三个分类都存在
- 每条都有来源与链接
- 文案没有会议稿/PPT 话术
- VuePress 导航已包含新页面
- `npm run lint` 通过
- `npm run build` 通过
- 代码已 push
- 最终返回的是站点地址 + 新闻简讯
